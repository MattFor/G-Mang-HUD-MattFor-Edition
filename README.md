# The quintessential spy main hud
I've been using it for over 15 years. Implemented fixes other people have made over the years, included some of my own, like the speedometer f.e, moved to the side as to not obscure the cloaks. Also, fixed the scoreboard, as it was trash on all of the other versions.

## Installation
1. Download the .zip from github
2. Put it in your TF2 install dir here: tf/custom/
3. unzip  

And that's it!
